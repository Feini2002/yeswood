# styles/

Split frontend styles. `index.html` loads `/styles/app.css`; root `/styles.css` redirects here for backward compatibility.

- `tokens.css` - design tokens
- `base.css` - reset, app shell, sidebar, typography
- `components.css` - shared UI (filter-bar, modals, tooltips, etc.)
- `pages/overview.css` - overview dashboard styles
- `pages/shared-pages.css` - shared panel, chart, status, and empty-state styles
- `pages/teams.css` - teams and owner-review styles
- `pages/profile.css` - franchise/direct profile dashboard styles
- `pages/details.css` - project workbench, drill modal, and project-detail styles
